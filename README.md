# DailyMate

DailyMate는 사용자의 목표를 이해하고 조언하며, 계획을 생성하고 캘린더에 반영하며 동기부여까지 제공하는
AI 비서형 시간관리 애플리케이션 입니다.

---

## 해결하려는 핵심 문제

### 1) 스스로 계획을 만들고 유지하는 능력 부족 해결
사람들이 루틴·습관 관리에서 가장 어려운 건 '설정'이 아니라 **의사결정의 부담**입니다.
매일 무엇을 먼저 할지, 얼마나 해야 하는지, 어떤 루틴을 넣을지, 캘린더에 어떻게 배치할지
스스로 결정해야 하는 일들이 쌓이면 결정 피로(decision fatigue)가 발생합니다.

**DailyMate는 이 부담을 제거합니다.**

### 2) 외부 시선(코치/조언자)이 있을 때 더 잘하는 문제 해결
사람은 누군가와 함께하면 더 꾸준히 행동합니다. DailyMate는 AI기술을 활용한 항상 옆에 있는 디지털 코치/비서 역할을 해 줌으로써 
사용자가 목표를 유지하고 행동을 지속하도록 도와줍니다.

### 3) 목표 → 일일 행동으로 연결되지 않는 문제 해결
큰 목표를 실제 행동으로 연결하는 것은 쉽지 않습니다. DailyMate는 AI와의 대화를 통해
"목표 → 루틴 → 오늘 할 일 → 캘린더 반영"의 흐름을 자동화하여 목표를 행동으로 변환합니다.

### 4) 루틴 설계의 번거로움 해결
루틴 고르기, 시간 정하기, 반복 설정, 우선순위 정리, 캘린더 반영 같은 설계 비용을 줄여
사용자가 생각하는 시간을 최소화합니다.

## 앱 내부 IA (Information Architecture)
아래는 DailyMate의 주요 탭 구조와 각 탭의 역할입니다.

- **Today** – 홈 / 오늘 대시보드
	- 오늘의 요약, 오늘의 목표(한 줄), 루틴 카드, 통합 타임라인, 빠른 AI 입력
- **Chat** – AI 비서와의 전체 대화 허브
	- 주간/일간 재설계, 목표 설정, 맥락 있는 플래닝 대화
- **Calendar** – 주·월 단위 일정/루틴 배치 뷰
	- 주간/월간 시각화, 드래그·드롭으로 편집, AI 재조정 요청
- **Stats** – 루틴/목표/집중시간 통계 & 회고
	- 달성률, 카테고리별 시간 사용량, 주간 리포트

- 메인 외 추가기능
  - Goals & Routines 설정
  - 계정/설정(Settings)
  - 온보딩 & 초기 목표 설정 플로우

## 시작하기
프로젝트를 빌드하고 실행하려면 Flutter SDK와 Xcode/Android Studio(플랫폼별)를 설정한 뒤:

```bash
flutter pub get
flutter run
```

자세한 개발 가이드와 아키텍처 문서는 `docs/` 폴더에 추가할 예정입니다.
